stylelint-config-alvinteh
=========

Configuration for stylelint.

Usage
-------

Add a `.stylelintrc` file to your project which extends `stylelint-config-alvinteh`. Example as follows:

```
{
  "extends": "styleint-config-alvinteh"
}

```
